# verified_orchestration

Research project workspace.

## Structure

- `paper/` - LaTeX paper draft and assets
- `experiments/` - experiment code, configs, data, and outputs

## Quick start

- Edit `paper/main.tex` for the manuscript.
- Put experiment scripts in `experiments/src/`.

## Build paper

From repo root:

```bash
make paper
```

Clean LaTeX outputs:

```bash
make paper-clean
```
